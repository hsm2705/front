* git 등록하기
1) 유저 등록하기
$ git config --global user.name "hsm2705"

2) 유저의 이메일주소 등록하기
$ git config --global user.email "hsm970210@gmail.com"

3) 등록 확인
$ git config --list

4) 명령어
git init

git add .

-- git status    // 현재 상태보기

git commit -m "first commit"

git remote add origin https://github.com/cheonbosik/front.git

-- git remote -v   // 잘 연결되었는지 확인...

git push origin master
